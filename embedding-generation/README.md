## Setup

Make sure to install the dependencies:

```bash
# pip
pip install -U google-generativeai 
pip install -U voyageai
pip install -U scikit-learn
pip install pandas

...

```
## Run
Run main.py

```bash
# go to embedding-generation/src directory
cd embedding-generation/src

```

```bash
# run 
python main.py
```
