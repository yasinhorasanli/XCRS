## Setup

Make sure to install the dependencies:

```bash
# pip
pip install google-generativeai 
pip install -U voyageai
...

```
## Run
Run main.py

```bash
# go to embedding-generation/src directory
cd embedding-generation/src

```

```bash
# run 
python main.py
```
